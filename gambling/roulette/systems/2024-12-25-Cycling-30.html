Since this is a "Couple's System" where you're literally betting on ALL 6 EVEN MONEY BETS... This grade is an EASY {% include ceg-system-rating.html rating="f" showRatingLetter=true showRatingName=false -%}.

If you must play, do it in single player mode.

When you win, remove the bet. When you lose, Martingale. You Martingale each bet independently, which make this system a {% include roulette-system-type.html type="Multi-Leg" %} system.

Reset when you clear the board. At this point, you'll be up {% include win-loss-amount.html amount=30 -%}.

As with most {% include roulette-system-type.html type="Martingale" %} + {% include roulette-system-type.html type="Multi-Leg" %} systems, you add MULTIPLE single points of failure. The downfall of Martingale is that you will inevitably hit a run of bad luck that wipes you out. Here, you're TRIPLING the chances that you hit that bad streak.
